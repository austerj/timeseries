timeseries.stats.mean(values)[source]¶

Return the sample mean of a numeric iterable.

Parameters

values – finite-length iterable of float-convertable numbers

Example

>>> import timeseries as ts

>>> values = (1, 2, 3)
>>> ts.stats.mean(values)
2.0
